# Frontend - Proyecto BD I

MVP en React + Vite para representar la plataforma de compatibilidad laboral descrita en el proyecto.

## Supuestos

- No hay conexión real con PostgreSQL por ahora.
- Los datos de `src/services/mockData.js` simulan tablas del esquema real: `usuario`, `perfil`, `profesional`, `empresa`, `experiencia`, `habilidad`, `ofertalaboral`, `posee`, `requiere`, `postula` y `certificadocurso`.
- La navegación se implementa con estado local para mantener el MVP simple y fácil de explicar.

## Comandos

```bash
npm install
npm run dev
```
